## Eligibility & Teams

- **Who:** Amrita University enrolled students (UG, PG, PhD)
- **Team Size:** 2–4 members (solo not allowed)
- **Registration:** Online via Nitrostack portal before 17 July (pre-registration required)
- **On-Site:** At least one team member must attend

---

## Project Requirements

**Build a functional MCP server that:**
- Implements **at least 2 of 3 primitives:** Tools, Resources, or Prompts
- Is **original work created during the hackathon** (no pre-existing projects)
- Uses **Nitrostack SDK/CLI** (or get written approval for alternatives)
- Is **deployed & publicly accessible** via Nitrostack platform
- Can use open-source libraries and third-party APIs (must be declared)

## Critical Rules

**Prohibited:**
- Plagiarism or code theft → immediate disqualification
- Dual team registration → all teams disqualified
- Disrupting other teams' infrastructure
- Undisclosed third-party code

**Permitted:**
- AI coding assistants (GitHub Copilot, Claude, ChatGPT) for implementation
- Problem definition and architecture must be your own

**Appeals:**
- Written submission within 30 minutes of winner announcement
- Reviewed by Chief Organiser + neutral faculty representative

## Success Tips

✅ **Come with a rough idea already in mind** — don't spend first hours deciding  
✅ **Focus on agent functionality** — build purpose-built MCP servers, not generic chatbots  
✅ **Integrate real data sources** — connect to live APIs, databases, or sensors  
✅ **Involve cross-disciplinary teams** — combine technical + domain expertise for better impact  
✅ **Document external dependencies** — declare all libraries and third-party code  
✅ **Deploy early and often** — test deployment on Nitrostack before submission deadline
